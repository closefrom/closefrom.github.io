第16章-直接定址表

2019-05-12
直接定址表（可依据数据，计算出所要找的元素的位置的表）
一个标号不但可以表示内存单元的地址，还可以表示内存单元的长度
如：
assume cs:code
code segment
a db 1,2,3,4,5,6,7,8
b dw 0
start:mov si,0
mov cx,8
s:mov al,a[si]
mov ah,0
add b,ax
inc si
loop s
mov ax,4c00h
int 21
code ends
end start
可见，code段中a，b标号后面没有：  他们同时表示内存地址和单元长度
如：b dw 0
指令：mov ax,b  相当于  mov ax,cs:[8]
指令：mov b,2  相当于   mov word ptr cs:[8],2
（特别地：后面有：的标号，只能在代码段中使用，不能在其他段中使用）
